## ğŸš€ Quick Start

### 1. Clone and Install

```bash
git clone <repository-url>
cd system-design-rag
npm install
```

### 2. Set Up Environment Variables

Create a `.env` file in the root directory:

```env
VITE_API_URL=/.netlify/functions
OPENAI_API_KEY=your_openai_api_key
PINECONE_API_KEY=your_pinecone_api_key
PINECONE_ENVIRONMENT=your_pinecone_environment
PINECONE_INDEX_NAME=system-design-rag
```

### 3. Collect and Process Data

```bash
# Collect system design content
npm run scrape

# Validate content quality
npm run validate

# Or run with detailed report
npm run validate:report
```

### 4. Process and Upload Data (Coming in Phase 2)

```bash
# Chunk content into optimal sizes
npm run chunk

# Generate embeddings
npm run embed

# Upload to Pinecone
npm run upload

# Or run all data processing steps
npm run data:all
```

### 5. Start Development Server

```bash
# Start Vite dev server
npm run dev

# Or start with Netlify Functions
npm run netlify:dev
```

## ğŸ› Troubleshooting

### Common Issues

**Data validation fails**
- Check JSON syntax in content files
- Ensure all required fields are present
- Run `npm run validate:report` for details

**API key errors**
- Verify `.env` file exists and has correct values
- Check API keys are valid and have proper permissions
- Ensure `.env` is not in `.gitignore`

**Build errors**
- Clear cache: `rm -rf node_modules && npm install`
- Update dependencies: `npm update`
- Check Node.js version: `node --version` (should be 18+)
